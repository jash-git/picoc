PicoC

This is a Win32 port of picoc (Pico C), a very small C language interpreter designed for UNIX-based systems.

Just type source code, save it to a file, and invoke the interpreter, specifying file name in the command line.

Works in Windows 2000, XP, 2003, Vista, Windows 7/8/10.
