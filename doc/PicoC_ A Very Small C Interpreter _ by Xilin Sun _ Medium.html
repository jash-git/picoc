PicoC: A Very Small C Interpreter

·

Repo

https://github.com/zsaleeba/picoc

Compile on Ubuntu

sudo apt install gcc make libreadline-dev
git clone https://github.com/zsaleeba/picoc.git
cd picoc
make
make test

Run

./picoc -i
picoc> int age = 23;
picoc> int phd = 10;
picoc> printf("%d\n", age + phd);
33

Kinda cool.
